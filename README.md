IMAGE CLASSIFICATION ON MINIST HANDWRITTEN DIGIT DATASET

Project Overview:

This project focuses on image classification using the MNIST Handwritten Digits dataset, which contains grayscale images of handwritten digits from 0 to 9. The goal is to build a machine learning model that can correctly identify the digit present in an input image.
The MNIST dataset is widely used as a beginner-friendly benchmark in machine learning and deep learning because of its simplicity and effectiveness in demonstrating classification techniques.

Dataset Description:

Dataset Name: MNIST Handwritten Digits
Number of Images: 70,000
60,000 training images
10,000 testing images

Color Format: Grayscale

Each image represents a single handwritten digit with a corresponding label.
